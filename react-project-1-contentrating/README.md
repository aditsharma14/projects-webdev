# Content Rating App

A simple React application that allows users to like or dislike a piece of content.  
It demonstrates basic React concepts such as state management and event handling, along with modern CSS styling.

---

## Features

- Like and Dislike buttons
- Live count updates
- Clean and modern UI
- Gradient background with glassmorphism effect
- Easy to understand React code

---

## Technologies Used

- React JS
- JavaScript (ES6)
- CSS3

How It Works

The app uses React state to track likes and dislikes

Clicking the buttons updates the count instantly

UI updates automatically without refreshing the page

Customization

You can:

Change the background colors in ContentRating.css

Replace static text with dynamic content

Add more features like reset, total votes, or animations
Purpose

This project is ideal for beginners who want to:

Learn React class components

Understand setState

Practice UI styling with CSS
